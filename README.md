# e_commer_project
# 👗 Myntra Fashion E-commerce Management System

## Project Overview

This project focuses on designing and developing a **Fashion E-commerce Management System** inspired by Myntra using Database Management System (DBMS) concepts. The main aim is to create a database that can efficiently manage customers, products, orders, payments, inventory, and delivery details.

The project will be developed in different phases throughout the semester. Each phase will cover important DBMS concepts such as database design, ER diagrams, normalization, SQL queries, and database implementation.

## Objectives

* Design a structured database for an online fashion store.
* Manage customer and product information efficiently.
* Maintain inventory and order details.
* Store payment and delivery records securely.
* Apply DBMS concepts to solve real-world business problems.

## Features 

* Customer Registration and Login
* Product Management
* Category Management
* Shopping Cart
* Wishlist
* Order Management
* Payment Management
* Inventory Tracking
* Order Tracking
